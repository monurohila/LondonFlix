# LondonFlix OTT Demo V19

Netflix-inspired independent OTT platform demo powered by UFP Film Club.

## Features
- LondonFlix branding
- Hero copy: Independent OTT Platform Powered By UFP Film Club
- Curated short films, full movies, web series demo and upcoming 2026 trailers
- Full movie trailer buttons
- Watch time/runtime badges
- Rich watch page details
- Search and filters
- My List using localStorage
- YouTube privacy-enhanced embeds with restricted app UI

## Vercel settings
Framework Preset: Other
Install Command: npm install --no-audit --no-fund --loglevel=error
Build Command: npm run build
Output Directory: public
